Реализовать чат в браузере со следующим функционалом:
-              просмотр списка активных комнат/активных личных чатов/пользователей
-              реализовать комнату/личный чат с подгрузкой истории последних сообщений
-              в случае комнаты должен присутсвовать список пользователей в комнате
-              создание/редактирование/удаление комнат
-              возможность общения в группе(комнате)/чат с пользователем
-              отображение различных нотификаций (например, новые сообщение). нотификация может быть представлена подсветкой комнаты/чата в списке
-              минималистический удобный UI (для быстроты и удобства можно воспользоватся bootstrap)

Реализовать сервис/модуль на клиенте который будет эмулировать вызов серверного API и асинхронность (использование setTimeout, setInterval).
Методы сервиса могут быть реализованы в виде: function (parameters, callback), где callback - function (err, data) {}
Либо можно реализовать методы, которые будут возвращать Promise.
Основные методы:
-              авторизация (проверка логина,пароля)
-              получение чатов/комнат
-              получение сообщений в чате
-              проверка новых сообщений
